Simple citizens feedback form.

== Description ==

Citizens Feedbacks for WordPress is a plugin for recieving feedback from citizens, plus the plugin provide 'Check status' form to check feedback's status.

== Installation ==

1. Upload the entire `citizens-feedbacks` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Citizens Feedbacks' menu in your WordPress admin panel.

== Documentation ==

To display 'Citizens Feedback' form add a shortcode [show_cfeedbacks_form] to a page.
To display 'Check feedback status' form add a shortcode [check_status] to a page.

== Frequently Asked Questions ==

= Is Citizens Feedbacks free? =

Yes, it is. All the functionality offered by the plugin is free, as long as you have a working WordPress installation.

= Can I translate Citizens Feedbacks in my language? =

Yes you can, by using Poedit. Go to 'languages' folder on the root of the plugin and create a translation from .pot file. For more help visit https://poedit.net/wordpress
